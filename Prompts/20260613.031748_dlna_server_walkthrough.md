# Walkthrough - DLNA/UPnP AV Media Server

The Java-based DLNA/UPnP AV Media Server has been successfully implemented.

## Changes Made

We created a complete Maven structure with production-quality code. The following packages and files were created under `src/main/java/com/vypeensoft/dlnaserver`:

## Verification

To compile the application, run tests, and package the executable:
```bash
mvn clean package
```

To run the application:
```bash
java -jar target/dlna-server.jar /path/to/your/media
```
Once launched, you can open VLC, navigate to Local Network -> Universal Plug 'n' Play, and double click "Java DLNA Server" to explore and play your media files.
